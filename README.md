## Features
- [x] File Manage
- [x] Folder Manage
- [x] File & Folder Share
- [x] Picture library
- [x] Storage Capacity Control
- [x] Support AwsS3,GoogleStorage,AliOSS,TencentCOS,QiniuKodo

## Install
```bash
curl -sSf https://dl.saltbo.cn/install.sh?k=zpan | sh
```

## QuickStart
```bash
zpan start
```

## Docker
```bash
docker run -p 80:8081 -v /etc/zpan:/root -it saltbo/zpan:latest
```
